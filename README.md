# React-News-App
A react news app which uses the newsdata.io api to fetch latest news.

# How to run the app?
```
1. Clone the repo.
2. Navigate to cloned repo in your PC.
3. Execute 'npm run dev' command.
```
# Technologies used: 
* ReactJS
* TailwindCSS 
* Vite 
* NPM 
* OpenWeatherMap API 
* Newdata API
